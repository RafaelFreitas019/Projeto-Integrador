Matemática - Funções Quadráticas

As funções quadráticas são um tipo de função polinomial do segundo grau. Sua forma geral é dada por: 

            f(x) = ax² + bx + c
Onde:
  • a: Coeficiente quadrático.
  • b: Coeficiente linear.
  • c: Termo constante.

Propriedades

  • Gráfico: O gráfico de uma função quadrática é uma parábola.
  • Raízes: As raízes da equação quadrática podem ser encontradas pela fórmula de Bhaskara.
  • Vértice: O vértice da parábola é dado por xv = -b/2a.

Exemplo

Considere a função f(x) = x² - 4x + 3. Vamos determinar as raízes:

  • Usando a fórmula de Bhaskara, temos:
    •                 x = (-(-4) ± √((-4)² - 4(1)(3))) / (2(1))
                x = (4 ± √(16 - 12)) / 2
                x = (4 ± √4) / 2
                x = (4 ± 2) / 2
    Portanto, as raízes são: x = 3 e x = 1.
